## 농협 스마트 경매 서비스 개발

* 기간
    - 2021 / 06 (Android 부분 1명)
    
* 역할
    - 안드로이드 어플리케이션 개발
    
* 참여도
    - 100%
    
* 주요 역량
    - Kotlin 활용 기술
    - AAC 활용 능력
    - MVVM 패턴 이해 능력
    - RxJava 활용 능력
    - Retrofit Http Library 사용
    - Glide Image Loader Library 사용
    - Hilt Library 사용
    - Netty 활용 능력
    - 메일을 통한 업무 공유 및 적극적인 협력 자세
