## 축협 스마트 경매 서비스 개발

* 기간
    - 2021 / 06 ~ 2021 / 10 (Android 부분 1명)
    
* 역할
    - 경매 어플리케이션 개발
    
* 참여도
    - 100%
    
* 주요 역량
    - Kotlin 활용 기술
    - Netty 프레임워크 활용 능력
    - AAC 활용 능력
    - MVVM 패턴 이해 능력
    - RxJava, RxKotlin 활용 능력
    - Retrofit Http Library 사용
    - Hilt Library 활용 능력
    - Google Cloud TTS 활용 능력
    - 카카오 i 라이브 활용 능력
    - 메일을 통한 업무 공유 및 적극적인 협력 자세
    
* 개발 내용
    - TCP, HTTP 를 활용한 경매 응찰 페이지 개발
    - 경매 제어 프로그램 TTS 개발
    - 실시간 방송 송출 화면 개발
    - 실시간 방송 수신 화면 개발
    
* 결과
    - 마감 기간 내 개발 완료 및 앱 런칭
